Category Creation Process:

  1. The first step was to break every description down to just individual words, and then count the occurences of each unique word in this list.
  2. Then a common word list was created from the words with more than 49 occurences.
  3. These common words were used to base the categories off.
  4. The categories are: SETS, Colors, Holidays, Material, Design, Jewelry, Vintage, Household
  5. Lists were then created containing the words that inspired the categories. These lists were used to display True if a description contained a word in the list for the matching category.
  6. After that, there was a column created to display True or False if the description contained a common word(words with more than 49 occurences).
  7. Finally all True and False occurences were converted to (1) and (0) respectively to total for a sum. Sum is used to create MISC categroy for descriptions not falling into any category.
